# gbrain — worker-агент

Маленькая программа, которая «оживляет» агента роя: берёт задачу с доски
(Supabase), выполняет её и отмечает результат.

## Как запустить

1. Установить зависимости (один раз):
   ```
   npm install
   ```

2. Создать файл `.env` из примера и вписать секретный ключ:
   ```
   cp .env.example .env
   ```
   Ключ `service_role` берётся в Supabase: **Project Settings → API → service_role**.

3. Запустить агента:
   ```
   npm run agent
   ```

Агент возьмёт одну задачу со статусом `todo`, отметит `in_progress` → `done`
и напишет в общий мессенджер роя.
