# gbrain — worker-агент

Маленькая программа, которая «оживляет» агента роя: берёт задачу с доски
(Supabase), выполняет её и отмечает результат.

## Как запустить

1. Установить зависимости (один раз):
   ```
   npm install
   ```

2. Создать файл `.env` из примера и вписать секретный ключ:
   ```
   cp .env.example .env
   ```
   Ключ `service_role` берётся в Supabase: **Project Settings → API → service_role**.

3. Запустить агента:
   ```
   npm run agent
   ```

Агент возьмёт одну задачу со статусом `todo`, отметит `in_progress` → `done`
и напишет в общий мессенджер роя.

## Мозг агента (LLM)

Агент умеет реально «думать» над задачей через **Groq**. Для этого впиши в `.env`:
```
GROQ_API_KEY=твой_ключ
```
Ключ берётся бесплатно на https://console.groq.com/keys.
Без ключа агент работает «без мозга» (заглушка) — это нормально для отладки каркаса.

## Два агента сразу

Окно 1: `npm run agent` (Takopi)
Окно 2: `AGENT_NAME=Архивариус npm run agent`
Поставить задачу: `node add-task.js "Название" "Описание" 1` (число — приоритет, меньше = важнее).
